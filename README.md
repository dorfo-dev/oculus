# Oculus

Oculus is a modern PDF viewer built with Electron and Vue 3. It allows users to view and interact with PDF files in a sleek and user-friendly interface.

## Features

- View PDF files with a modern interface
- Cross-platform support (Windows, Linux, macOS)
- Open-source and customizable

## Installation

### Prerequisites

- Node.js (v18 or higher)
- npm (v8 or higher)

### Clone the repository

```bash
git clone https://github.com/dorfo-dev/oculus.git
cd oculus
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
npm run electron:dev
```

## Build

To build the application for production:

```bash
npm run electron:build
```

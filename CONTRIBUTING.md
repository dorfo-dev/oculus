# Contributing to Oculus

Thank you for considering contributing to Oculus! We welcome contributions from the community to make this project better.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Make your changes and commit them with clear messages.
4. Push your branch to your fork.
5. Open a pull request to the `main` branch of this repository.

## Code of Conduct

Please follow our [Code of Conduct](CODE_OF_CONDUCT.md) to ensure a welcoming environment for everyone.

## Issues

If you find a bug or have a feature request, please open an issue in the repository. Provide as much detail as possible to help us address the issue quickly.
